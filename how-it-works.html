How to Use These Gemini AI Photo Prompts in Nano Banana

  1. Open Gemini (web or app).
  2. Choose Create image from the tools menu.
  3. Pick the model:
    • Fast → Nano Banana
    • Thinking → Nano Banana Pro
  4. Paste one of the prompts below (remove backticks if you copy from a code block).
  5. Where a prompt says Upload, first add your reference image(s), then run the prompt.
  6. If the first result isn’t quite right, use the Troubleshooting section further down to iterate instead of starting from scratch.
